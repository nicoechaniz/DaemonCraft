mBit Architecture

DaemonCraft Perception Layer — Minecraft chunks as LLM-native text

Perception Layer

5 formats from 1 world:

  • Binary (256b) — navigation
  • Columns (256 pairs) — terrain
  • Rows (1536 nums) — horizon
  • Surface (256c) — ground type
  • Full (4096c) — exact world

Cognition Layer

LLM receives text-native world state:

  • Pattern recognition (paths, caves)
  • Spatial reasoning (distances, volumes)
  • Format selection per task
  • Decision: direct action vs delegation

Verification Loop

Every action is verified:

  • perceive_before → act → perceive_after
  • State diff confirms success/failure
  • Retry with different approach if needed
  • mBit as ground truth
