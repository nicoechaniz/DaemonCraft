# Commission Brief: DaemonCraft Custom UI Frame Set

**Platform:** Fiverr / DeviantArt / r/HungryArtists
**Budget:** $75-150
**Deadline:** 3 weeks from acceptance

---

## Deliverables

1. `textures/gui/widgets.png` — button frames, slider, hotbar highlights
2. `textures/gui/title/background.png` — title screen panorama or static bg
3. `textures/gui/container/inventory.png` — inventory GUI border

## Style brief

- **Palette:** Cyan `#2ec4b6` as accent, dark grey `#1a1a2e` as base, purple `#7b2cbf` for highlights
- **Style:** Vanilla-faithful — should look like a natural extension of the default GUI, not a complete genre shift
- **Mood:** Clean, readable, slightly futuristic but not sci-fi

## Technical requirements

- Widget sheet must align to the standard Minecraft GUI grid (refer to vanilla `widgets.png` layout)
- Inventory border must be 256x256 with 16x16 cell grid
- Title background: 1024x1024 minimum, seamless if tiled
- Deliver PNGs + layered sources
