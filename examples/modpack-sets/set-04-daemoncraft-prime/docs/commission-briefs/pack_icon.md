# Commission Brief: DaemonCraft Pack Icon

## Overview
256×256 pixel-art icon for the DaemonCraft Minecraft server. Used in the
Bedrock resource pack picker and as the Java pack thumbnail.

## Budget
$50-100 USD

## Deliverables
- 256×256 PNG with transparent background
- 128×128 downscale (bonus, if artist provides)
- 64×64 favicon-style (bonus, if artist provides)
- Source file (.ase, .psd, or .svg)

## Style direction
**Vanilla-faithful pixel art.** 16-bit aesthetic, limited palette (~16 colors),
crisp edges, no anti-aliasing.

## Color palette
- Primary: `#2ec4b6` (cyan — the DaemonCraft brand color)
- Secondary: `#9b5de5` (purple — accent/magic)
- Background: transparent (or pure black `#000000` if artist prefers)
- Highlights: `#f15bb5` (pink — optional accent)

## Concept
A stylized daemon/portal motif — something that reads as "magical gateway" or
"friendly spirit" at 256×256. Not scary. Kid-friendly.

**Reference concepts** (AI-generated, included as vibe guides only):
- See `docs/ai-concepts/pack_icon_concepts.png`
- Preferred direction: concept #3 (cyan gem with purple glow, geometric)

## Constraints
- Must read clearly at 64×64 (server list thumbnail size)
- No text/letters (text is illegible at small sizes)
- No gradients — flat colors or dithering only
- Must be original work — no Minecraft IP (creeper face, Mojang logo)

## Timeline
- Concepts: 3-5 days
- Revisions: 2-3 days
- Final delivery: 7 days total

## Rights
Work-for-hire, all rights assigned to DaemonCraft Project. Credit in
`CREDITS.md`. Right to redistribute under CC-BY-SA-4.0.

## Payment
50% upfront on concept approval, 50% on final delivery + signed rights
assignment.
